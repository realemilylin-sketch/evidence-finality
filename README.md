# Evidence Finality

**Evidence Finality** is a reference definition for when evidence becomes
stable, attributable, and effectively non-revisable in dispute contexts.

This repository is a **public reference mirror** of the documentation site:
- Primary reference: https://evidencefinality.com

## What this is
- A conceptual definition and vocabulary for “finality” in evidence
- Reference cases and minimal evidence sets (descriptive, not prescriptive)
- Non-normative documentation for citation and clarity

## What this is not
- Not legal advice
- Not a product or service
- Not a normative standard

## Revision policy
Slow, structure-preserving updates.
